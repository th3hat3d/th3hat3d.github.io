HTB - Noter Writeup

I quite enjoyed this one. Noter centers around a web application with a weak signing key for its session cookies, which allows us to impersonate a user and access a backup of the website source code over FTP. Upon inspecting the source, we find that the web app uses a javascript library with an RCE vulnerability. Once on the box, we find out that MySQL runs as root, and is therefore vulnerable to code execution as root with a user defined function (UDF) using the raptor exploit.
